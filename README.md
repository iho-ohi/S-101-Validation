# S-101-Validation
S-101 Github repository for maintenance of S-158:101.

Comments and suggestions for improvement of the checks are highly appreciated. Please include the check ID in the title of each issue that relates to an existing check.

## Description of folders<br>
### Old working files<br>
This folder contains older working files that have previously been uploaded to this repository.
<br><br>
### Published versions<br>
This folder contains the published version of S-158:101, as well as redline files that describe the differences between versions. NOTE: The authoritative files for the official version are the files that can be found in the GI Registry.
